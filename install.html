شایان ذکر است در آموزش داده شده توجه نمایید تمام فایل ها را با نرم افزار ویرایشگر کد به غیر از Frontpage باز نمایید.

پس از اینکه از حالت ZIP فایل را خارج کردیم تمامی محتویات را عینا به Root محل نصب سیستم انتقال می دهیم تا فایل و پوشه ها جایگزین گردند. سپس مراحل زیر را انجام می دهیم.
توجه :
شاخه templates را باز نموده فایل های محتوی شاخه yourtheme را به شاخه قالب خودتان انتقال دهید.

مرحله اول : فایل install-vip.php را اجرا نمایید. تا سیستم نصب شود

مرحله دوم : فایل engine.php را باز نمایید:
به دنبال کد زیر بگردید:

case "addnews" :
include ENGINE_DIR . '/modules/addnews.php';
break;

مرحله سوم: کد زیر را بعد از کد بالا قرار دهید:

case "vip_user" :
if ($is_logged) {
$config['allow_cache'] = false;
include ENGINE_DIR . '/modules/vip_user.php';
} else
msgbox ( $lang['all_err_1'], $lang['fav_error'] );
break;
